Getting Started with Tailwind CSS

Tailwind CSS is a utility-first CSS framework that lets you use pre-built utility classes right in your HTML.

In simple terms: it speeds up our web developing!

Installing Tailwind CSS

There are more appropriate ways of installing Tailwind for a real website, but for our purposes, all we need to do is add a single link in our header tag.

Feel free to copy all of this code to use as a starting template for your projects:

HTML Boilerplate with Tailwind CSS
<!DOCTYPE html>
        <html lang="en">

        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>My Tailwind Project</title>
            <script src="https://cdn.tailwindcss.com"></script>
        </head>

        <body>
            <div class="container mx-auto p-4">
                <h1 class="text-3xl font-bold">
                    Hello Tailwind!
                </h1>
            </div>
        </body>

</html>

Tailwind vs Traditional CSS

We still need to understand the basics of traditional CSS in order to get the most out of Tailwind, but it can help speed up our developing a lot.

Let's see how Tailwind CSS compares to traditional CSS with some common examples:

Traditional Tailwind
Traditional CSS - Header
<style>
.big-header {
    font-size: 2.25rem;
    font-weight: bold;
    margin-bottom: 1rem;
}
</style>

<h1 class="big-header">
    Hello World
</h1>
Tailwind CSS - Header
<h1 class="text-4xl font-bold mb-4">
    Hello World
</h1>

Hello World

Traditional Tailwind
Traditional CSS - Button
<style>
.cool-button {
    background-color: #3b82f6;
    color: white;
    padding: 0.5rem 1rem;
    border-radius: 0.25rem;
    transition: background-color 0.3s;
}
.cool-button:hover {
    background-color: #2563eb;
}
</style>

<button class="cool-button">
    Click Me
</button>
Tailwind CSS - Button
<button class="bg-blue-500 text-white px-4 
py-2 rounded hover:bg-blue-600 
transition-colors">
    Click Me
</button>
Traditional Tailwind
Traditional CSS - Card
<style>
.card {
    background: white;
    border-radius: 0.5rem;
    padding: 1.5rem;
    box-shadow: 0 4px 6px -1px rgba(0,0,0,0.1);
}
.card-title {
    font-size: 1.25rem;
    font-weight: bold;
    margin-bottom: 0.5rem;
}
</style>

<div class="card">
    <h3 class="card-title">Card Title</h3>
    <p>Card content goes here...</p>
</div>
Tailwind CSS - Card
<div class="bg-white rounded-lg p-6 
shadow-lg dark:bg-gray-800">
    <h3 class="text-xl font-bold mb-2">
        Card Title
    </h3>
    <p>Card content goes here...</p>
</div>

Card Title

Card content goes here...

Notice the Difference?

  • Way less code: No separate CSS files or style tags needed
  • Everything in one place: Style directly in your HTML elements
  • Responsive design: values like md: and lg: automatically respond to your screen size
