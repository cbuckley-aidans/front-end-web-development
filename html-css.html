HTML Basics

Document Structure

Every HTML page needs a basic structure. The head section contains metadata, while the body contains visible content.

Code
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Web Page</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>

Headings

HTML headings create a clear hierarchy of importance. Remember that 1 is the largest, and 6 is the smallest.

Code

Paragraphs and Text Formatting

Paragraphs are the building blocks of text content. Use semantic tags for emphasis and strong importance.

Code
<p>This is a regular paragraph of text.</p>

<p>
    This paragraph has <em>emphasized</em> and 
    <strong>important</strong> text.
</p>

<p>
    Text can be <mark>highlighted</mark>,
    <sup>superscript</sup>, or
    <sub>subscript</sub>.
</p>

Tables

Tables organize data into rows and columns. Use semantic table elements for better accessibility.

Code
<table>
    <caption>Monthly Sales</caption>
    <thead>
        <tr>
            <th scope="col">Month</th>
            <th scope="col">Sales</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>January</td>
            <td>$10,000</td>
        </tr>
        <tr>
            <td>February</td>
            <td>$12,500</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <th scope="row">Total</th>
            <td>$22,500</td>
        </tr>
    </tfoot>
</table>

CSS Basics

Classes and Selectors

CSS selectors target elements for styling. Classes allow reusable styles across multiple elements.

Code
/* This would apply to ALL paragraph tags. */
p {
    line-height: 1.6;
}

/* This would ONLY apply to tags that have the class "highlight". */
.highlight {
    background-color: yellow;
    padding: 0.5em;
}

/* ID Selector - IDs are unique and only used for ONE thing on the page. */
#main-header {
    font-size: 2em;
    margin-bottom: 1em;
}

/* ALL paragraph tags that belong to the card class only. */
.card p {
    margin-bottom: 1em;
}

/* ALL tags of a certain kind that are hovered over with the mouse. */
a:hover {
    color: #ff0000;
}

.button:focus {
    outline: 2px solid blue;
}

Box Model

The CSS box model defines how elements take up space with content, padding, border, and margin.

Code
.box {
    /* Content dimensions */
    width: 200px;
    height: 100px;

    /* Inner spacing */
    padding: 20px;
    
    /* Border */
    border: 2px solid #666;
    border-radius: 8px;
    
    /* Outer spacing */
    margin: 20px;
}

.box-sizing {
    box-sizing: border-box;
}

Forms and Input

Form Controls

Create accessible forms with proper labels and input types.

Code
<form>
    <div class="form-control">
        <label for="name">
            Name: <span aria-hidden="true">*</span>
        </label>
        <input 
            type="text" 
            id="name"
            required
            placeholder="Enter your name"
        >
    </div>

    <div class="form-control">
        <label for="email">Email:</label>
        <input 
            type="email" 
            id="email"
            placeholder="you@example.com"
        >
    </div>

    <fieldset>
        <legend>Preferences:</legend>
        <div class="checkbox">
            <input 
                type="checkbox" 
                id="newsletter"
            >
            <label for="newsletter">
                Subscribe to newsletter
            </label>
        </div>
    </fieldset>

    <button type="submit">Submit</button>
</form>

Animations and Transitions

CSS Transitions

Smooth transitions between element states enhance user experience.

Code
.transition-all {
    transition: all 0.3s ease;
}

.hover-transform:hover {
    transform: scale(1.1) rotate(5deg);
}

.hover-colors:hover {
    background-color: #4f46e5;
    color: white;
}

.hover-shadow:hover {
    box-shadow: 0 0 15px rgba(59, 130, 246, 0.5);
}
Example

Keyframe Animations

Create complex animations using @keyframes rules.

Code
@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-20px);
    }
}

.bounce {
    animation: bounce 2s infinite;
}

@keyframes pulse {
    0% {
        transform: scale(1);
        opacity: 1;
    }
    50% {
        transform: scale(1.1);
        opacity: 0.7;
    }
    100% {
        transform: scale(1);
        opacity: 1;
    }
}

.pulse {
    animation: pulse 2s infinite;
}
