DaisyUI: Making Tailwind Even Easier

DaisyUI is a component library for Tailwind CSS that provides pre-built components with semantic class names.

Instead of writing class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded", you can simply write class="btn btn-primary"!

Why Use DaisyUI?

It speeds up development, makes code more readable, and provides consistent styling across your project.

Getting Started

Here's your starter template. Just copy this code into a new HTML file and you're ready to go!

HTML Template 
<!DOCTYPE html>
<html lang="en" data-theme="dark">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My DaisyUI Project</title>
    <link href="https://cdn.jsdelivr.net/npm/daisyui@4.12.23/dist/full.min.css" rel="stylesheet" type="text/css" />
    <script src="https://cdn.tailwindcss.com"></script>
</head>
<body>
    <div class="container mx-auto p-4">
        <h1 class="text-3xl font-bold">
            Hello DaisyUI!
        </h1>
        <button class="btn btn-primary">Click me</button>
    </div>
</body>
</html>

Theme Support

Notice the data-theme="dark" attribute? DaisyUI comes with built-in themes!

Form Elements

Form Examples 
<form class="form-control w-full max-w-xs">
    <!-- Text Input -->
    <label class="label">
        <span class="label-text">Username</span>
    </label>
    <input type="text" 
           placeholder="Type here"

        <option>Harry Potter</option>

    </select>

    <!-- Checkbox -->
    <div class="form-control">

            <input type="checkbox" 
                   class="checkbox" />
        </label>
    </div>

    <!-- Radio Group -->
    <div class="form-control">
        <label class="label cursor-pointer">
            <span class="label-text">Red</span>
            <input type="radio" 
                   name="radio-10" 
                   class="radio" 
                   checked />
        </label>
    </div>

    <!-- Toggle -->

            <span class="label-text">
                Dark Mode
            </span>

    </div>

    </label>

Modal Dialogs

Create accessible modal dialogs with DaisyUI:

Modal Examples 
<!-- Button to trigger modal -->
<button class="btn" onclick="my_modal.showModal()">
    Open Modal
</button>

<!-- Modal Dialog -->
<dialog id="my_modal" class="modal">
    <div class="modal-box">
        <h3 class="font-bold text-lg">
            Hello Modal!
        </h3>
        <p class="py-4">
            Press ESC key or click outside to close
        </p>
        <div class="modal-action">
            <form method="dialog">
                <button class="btn">Close</button>
            </form>
        </div>
    </div>
    <form method="dialog" class="modal-backdrop">
        <button>close</button>
    </form>
</dialog>

<!-- Modal with Multiple Actions -->
<button class="btn btn-primary" 
        onclick="action_modal.showModal()">
    Modal with Actions
</button>

<dialog id="action_modal" class="modal">
    <div class="modal-box">
        <h3 class="font-bold text-lg">
            Confirm Action
        </h3>
        <p class="py-4">
            Are you sure you want to proceed?
        </p>
        <div class="modal-action">
            <form method="dialog" class="flex gap-2">
                <button class="btn btn-error">
                    Cancel
                </button>
                <button class="btn btn-primary">
                    Confirm
                </button>
            </form>
        </div>
    </div>
    <form method="dialog" class="modal-backdrop">
        <button>close</button>
    </form>
</dialog>

Customizing Components

You don't need to settle for the provided component and theme. You can use Tailwind CSS to customise components to your liking without doing all the original heavy lifting.

Customization Example 
<!-- Basic DaisyUI button -->
                <button class="btn btn-primary">
                    Basic Button
                </button>
                
                <!-- Customized with Tailwind -->
                <button class="btn btn-primary 
                    shadow-lg transform hover:scale-110 
                    transition-all duration-300 
                    rounded-full px-8">
                    Enhanced Button
                </button>

Added Tailwind classes:

  • shadow-lg - Adds shadow
  • transform & hover:scale-110 - Grow on hover
  • rounded-full - Full rounded corners
  • px-8 - Extra horizontal padding
